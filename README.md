## 1、说明
在esp32上实现简易的声音频谱，只用到两个库：arduinoFFT和adafruit SSD1306。

## 2、bom：

esp32-wroom-32，max4466，0.96OLED

## 3、接线：

OLED：
SCL->P21
SDA->P22

MAX4466：OUT->P0
